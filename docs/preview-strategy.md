# Preview Strategy

Pass 1 does not build a visual preview UI.

For later passes, preview means:

- curated reference image;
- role map for the look;
- tutorial framing and application steps;
- product-role readiness from the backend.

Preview does not mean AR try-on, face tracking, makeup filtering, camera shade detection, or a claim that a static overlay is an accurate personal result.

Any later visual placeholder must be labeled as a reference or demo view, not as a live try-on.
